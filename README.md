# Расширение stack-lang для Visual Studio Code

Добавляет возможности для встроенного языка Комплекса программ "СТЕК" (_.prg, _.hdl, *.rpt)

---

## Возможности
- Подсветка синтаксиса
- Навигация по коду
- Сниппеты (шаблоны текста)
- Форматирование кода
- Валидатор кода ( в разработке )
